# CareMaster React Native Development Test

## Environment
Node version: 18.14.0
npm version: 9.3.1

## Run the iOS project
```
npm install
```
```
cd ios && RCT_NEW_ARCH_ENABLED=1 pod install
```
```
cd ..
```
```
npx react-native run-ios
```


## Run the android project
```
npm install
```
```
npx react-native run-android
```
